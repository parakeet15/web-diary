基本的な使い方

  • 日記を作成する
  • 日記を保存する
  • 日記を削除する
  • 日記を検索する
  • ファイルを添付する

日記を作成する

をクリックして日記を作成できます。

ショートカットキー
Windows (Chrome) Mac (Chrome)
Alt + Shift + C Control + Option + C

日記を保存する

をクリックして日記を保存できます。
ローカルストレージの空き容量が不足すると保存できません。

ショートカットキー
Windows (Chrome) Mac (Chrome)
Alt + Shift + S Control + Option + S

日記を削除する

をクリックして日記を保存できます。

ショートカットキー
Windows (Chrome) Mac (Chrome)
Alt + Shift + X Control + Option + X

日記を検索する

をクリックして日記を検索できます。

ショートカットキー
Windows (Chrome) Mac (Chrome)
Alt + Shift + F Control + Option + F

ファイルを添付する

をクリックしてファイルを添付できます。
ファイルサイズは1MBに制限されます。

ショートカットキー
Windows (Chrome) Mac (Chrome)
Alt + Shift + O Control + Option + O
